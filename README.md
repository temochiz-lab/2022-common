# このアカウントについて
jsPsych ver.7 を使った実験のプログラム達です。

# jsPsych の使い方
いくつかの使い方がありますが、  
・Internet接続環境が用意できない場合がある  
・バージョンを固定しておきたい  
という理由で、ローカルのPCにコピーした状態で使うのを前提にしています。

# jsPsych の入手方法
本家 https://github.com/jspsych/jsPsych/releases から、Dist archive (zip) を、ダウンロードしてください。
