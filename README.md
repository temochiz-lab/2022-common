# このアカウントについて
jsPsych ver.7 を使った実験のプログラム達です。

# jsPsych の使い方
いくつかの使い方がありますが、  
・実験室のPC用に Internet 接続環境を用意できない場合がある  
・バージョンを固定しておきたい  
という理由で、ローカルのPCにコピーした状態で使うのを前提にしています。
